# 🎨 공동 작업 그림일기 (Static Edition)

Cloudflare Pages와 같은 정적 호스팅 환경에 최적화된 **공동 작업 그림일기**입니다.

## ✨ 주요 기능
- **정적 배포 최적화**: 서버 없이 `index.html`만으로 동작하는 가벼운 구조입니다.
- **달력 기반 조회**: 과거 날짜 클릭 시 `resources/` 폴더 내의 정적 파일들을 불러옵니다.
- **LocalStorage 저장**: 오늘 작성한 일기는 브라우저 로컬 저장소에 보관됩니다.
- **아날로그 감성**: 원고지 스타일의 입력 칸과 타이핑 애니메이션을 제공합니다.

## 🚀 배포 및 실행
1. GitHub 저장소를 Cloudflare Pages에 연결합니다.
2. 빌드 설정 없이 루트 디렉토리의 `index.html`이 자동으로 배포됩니다.

## 📁 데이터 관리 (resources/)
과거 일기를 미리 표시하고 싶다면 `resources/` 폴더에 아래 형식으로 파일을 업로드하세요.
- 그림: `YYYY-MM-DD_그림.jpg`
- 내용: `YYYY-MM-DD_내용.txt`
